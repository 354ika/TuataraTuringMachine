How to start the software
-------------------------

Ensure that you have the java JRE (or JDK) version 1.5 or later installed.
tuataraTMSim.gui.MainWindow is the main entry point to the program.  After
compiling, this should be called via

> java tuataraTMSim.gui.MainWindow

at the command prompt (Windows and Linux).
